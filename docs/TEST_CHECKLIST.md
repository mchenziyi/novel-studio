## 五、建议补充的测试

### 优先级高

1. **Playwright E2E 测试**: 覆盖章节编辑器的浏览器端渲染
2. **CSS 布局回归测试**: 使用 Playwright 截图对比
3. **API 集成测试**: 端到端 API 调用链测试

### 优先级中

4. **组件测试**: ChapterEditor, AiEditSidebar 的完整交互测试
5. **错误处理测试**: 网络错误、API 超时、数据库异常
6. **版本历史边界测试**: 大量版本、并发保存

### 优先级低

7. **性能测试**: 大章节加载、大量版本查询
8. **响应式测试**: 不同屏幕尺寸下的布局
9. **可访问性测试**: 键盘导航、屏幕阅读器
